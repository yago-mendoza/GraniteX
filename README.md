# GraniteX

A 3D CAD application built in Rust.

## Status

Early development — building the core rendering and geometry engine.

## Requirements

- Rust 1.75+ (install via [rustup](https://rustup.rs))
- GPU with Vulkan, DX12, or Metal support

## Getting Started

```bash
cargo run
```
